# 配置页

Api2Code 提供了功能丰富的配置页，帮助更好地定制你的功能。

# 打开配置页

# 自动打开

当插件 完成安装 时，默认会打开 配置页。

# 手动打开

当需要手动打开配置页时，右击 插件图片，左键点击 选项 打开 配置页面。

# 配置页概览

# 更新配置

修改配置页信息后，点击 保存按钮，即可使修改后的配置生效。

提示

请关闭配置页前，务必记得保存，否则修改记录可能会丢失。
